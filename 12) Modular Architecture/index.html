Modular Architecture

Functions of each module

Abstract Module Functions
Movement Module Determines the movement of individual players around the board and tracks the location of the players on the board at the end of the turn.
Objective Tracking Module Tracks how close each player is to their respective goals (the number of objective tokens each player has)
Chance Card Module Enforces actions which influences the playerâ€™s pieces, cards and/or tokens as well as allowing players to interact with each other.
Mission Square Module Interacts with player pieces. Allows players to trade in cards for tokens if a criteria is met.
Board Module Governs the actions of the players in accordance to the game rules perimeters set on the board
